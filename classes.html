Classes

                    
    class Shape {
        int numberOfSides = 0
        string fun simpleDescription() { // notice class can also have funtions
            return "A shape with [numberOfSides] sides."
        }
    }

Class are declared very similar to functions. 

                    
        personalInfo Sarah
        Sarah.name = "sarah h"
        Sarah.age = 19
        int youngest = Sarah.age

Classes can also be edited by using dot notation. 

                    
        class car {
            string make
            string model 
            int year
            
            def (string make, string model, int year) {
                self.make = make
                self.model = model
                self.year = year
            }
        }

        car dadCar("Toyota", "Camary", 2012)

Classes can also have a definer function that can be used to set values as soon as value is called. 

                    
        struct Card {
            string Rank
            string Suit
        }
            
        Card threeOfSpade= Card("three", "spades")

If the user wants to have a constant class, then they can use a struct.

Unlike classes, structs don't need definers like functions. They are already declared in order.
