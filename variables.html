Variables of the same type can be combined with plus and combined variables can be implicit. 

                    
    int orange = 10
    double apple = 10.5
    double fruits = double(orange) + apple //20.5

Variables can be redeclared when combined. 

                    
    print: "I have 'fruits' fruits." // I have 20.5 fruits
