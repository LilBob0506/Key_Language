Functions are declared type followed by "fun" then name. Type is void if not specified. These are displayed above. 

                    
    int fun addOne(int num) {
        return num++
    }
    int fun timeTwo(int num) {
        num*2
    }

    int number = 5
    number = addOne(number) + timeTwo(number) // number = 12

Functions can also be added together going from left to right. 

                    
    class personalInfo {
        string name
        int age
    }
        
    class fun printPersonal(personalInfo person, string fullName, int age) {
        person.name = fullName
        person.age = age
        return person
    }
        
    personInfo newPerson
    newPerson = printPersonal(newPerson, "John Smith", 20) // newPerson.name = John Smith newPerson.age = 20

Functions can also return classes.

Notice you don't have to label variables when you call a function. It automatically goes in the same order as the declaration.

Functions can call any other function that is declared in the code order doesn't matter 

                    
    S fun highest (int Values) {
        int check = 0
        for Values till Values.end{
            if check < Values  check = Values
        }
        print: "[check] is highest value."
    }
    S fun lowest (int Values) {
        int check = 0
        for Values till Values.end {
            if check > Values  check = Values
        }
        print: "[check] is lowest value."
    }
    
    int values = [10,15,60,75,600,1]
    split S // create a split variable and all variables of this name can be done at the same time
    S highest(values)
    S lowest(values)
    combine S // wait for all s variables to finish before moving on

In Key, you can run multiple functions. If you want to run multiple functions at the same time you can use split and combine as shown above.
