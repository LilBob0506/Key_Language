Key Programming Language

Key is an innovative and efficient new way to develop code. Key utilizes various commands to help make coding moreconvenient to developers.

These commands include variable and function additions (ie. fun + fun and string + string). The purpose of these commands in Key is to remove code redunancy.

It takes what makes languages like C and C++ and makes it more efficient. It also makes sure to auto initialize declares so you dont have to worry about zeros and nil.

It also helps with loops by making the end of it able to be used as in the loop so you dont have to memorize the size of anray if it's large. Also iff gives you more options for bool.
